# 使用说明

## 1. 连接本地模型

点击页面顶部的 `连接本地模型`，可以配置：

- 默认引擎
- Stable Diffusion WebUI 地址
- LoRA 模型选择
- ComfyUI 地址
- ComfyUI checkpoint
- ComfyUI workflow 文件路径

保存后，网页生成会直接使用新的后端配置。

## 2. 输入提示词

推荐两种用法：

### 方式 A：中文自然语言

例如：

```text
一张明式圈椅放在素雅茶室中，黑胡桃木，榫卯结构，晨光斜照，留白克制，高级产品摄影
```

系统会自动：

- 识别中文
- 翻译英文关键词
- 做中式家具场景增强

### 方式 B：结构化英文 Prompt

如果你已经有成熟的 SD Prompt，直接粘贴即可。

系统会尽量原样保留，不会过度强化。

## 3. 查看三段式提示词

在 `AI 优化提示词` 面板里可以看到：

- 中文原始输入
- 自动翻译英文 Prompt
- 强化后 Prompt

这是排查“翻译有问题”还是“增强有问题”的核心区域。

## 4. 选择基础参数

目前前端保留了几个高频控制项：

- 风格预设
- 精细度
- 画布尺寸
- 空间氛围

默认更偏“稳出图”而不是“高参数拉满”。

## 5. 生成后操作

每张结果卡片支持：

- 放大
- 下载
- 再来一版

“再来一版”支持：

- 同参数重跑
- 换个角度
- 更强光影
- 更克制陈设

## 6. 继续细化上一版

生成过至少一张图后，会出现“继续细化上一版”卡片：

- 显示上一版缩略图
- 显示上一版描述
- 一键带回输入框继续迭代
